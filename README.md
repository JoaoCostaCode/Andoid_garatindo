# App Músicas do Garantido

Um aplicativo Android simples e interativo para celebrar as toadas do **Boi Garantido**, o boi do povão do Festival Folclórico de Parintins!

## 📝 Sobre o Projeto

O aplicativo foi desenvolvido com o objetivo de ser um projeto de aprendizado prático em desenvolvimento Android, unindo conceitos de manipulação de estados e eventos de clique com a cultura amazonense.

Ao abrir o aplicativo, o usuário visualiza uma tela inicial com a descrição padrão **"Garantido"**. Ao interagir com o botão principal, o aplicativo alterna dinamicamente o texto, exibindo os nomes das toadas do boi vermelho e branco.

##Funcionalidades

**Interatividade Simples:** Um botão centralizado que responde ao clique do usuário.
**Alteração de Estado Dinâmica:** Atualização textual em tempo real sem mudar de tela.
**Seleção de Toadas:** Lista interna contendo clássicos do Garantido.

## Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias e ferramentas:

**Linguagem:** Kotlin 
**Interface Visual:** XML / ViewBinding 
**IDE:** Android Studio
**SDK Mínima Suportada:** Android 7.0 (Nougat - API 24)
**SDK Alvo:** Android 16 (API 36)


## Como Executar o Projeto

Para testar o aplicativo na sua máquina ou no seu celular físico, siga os passos abaixo:

### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:

* Android Studio (versão atualizada)
* Celular android ou emulador: um dispositivo físico Android com o **Modo Desenvolvedor** e a **Depuração USB** ativados, ou um emulador configurado no Android Studio

### Passo a Passo

#### 1. Clone este repositório

```bash
git clone https://github.com/JoaoCostaCode/Andoid_garatindo.git
```

#### 2. Abra o projeto no Android Studio

* Abra a IDE Android Studio
* Vá em **File > Open...**
* Selecione a pasta onde você clonou o projeto
* Aguarde o Gradle baixar as dependências e sincronizar o projeto

> Na primeira execução isso pode levar alguns minutos.


#### 3. Rode o aplicativo

* Selecione seu dispositivo físico ou emulador
* Clique no botão **Run**
